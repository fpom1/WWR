Image Encryptor

Description

This is a python program that can take an image, and generate a random key to encrypt your image. The new encrypted image and the key will be saved so, using this program, you can also decrypt your image. This project isnt very user friendly, so I wouldnt reccomend using it without some coding knowlege.

Instructions

Instructions on how to use this program are provided in the comments of the python file. Make sure you understand them before running the program.
